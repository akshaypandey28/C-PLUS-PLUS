- Prim's Algorithm: Finds the Minimum Spanning Tree (MST). It connects all nodes in a graph with the lowest possible total edge weight, ensuring no cycles.

- Dijkstra's Algorithm: Finds the Shortest Path from a single source to all other nodes. It minimizes the cumulative weight from the starting point to each destination.
